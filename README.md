# Task Engineering App

Aplicación web para gestión de tareas de ingeniería de software.

## Descripción

Task Engineering App permite registrar, visualizar y eliminar tareas con distintos niveles de prioridad (alta, media, baja). Desarrollada como práctica de GitHub Flow en la asignatura **Desarrollo Web Integral** — UTM, 9D, Mayo 2026.

## Estructura del proyecto

```
task-engineering-app/
│
├── index.html       # Página principal con formulario de tareas
├── about.html       # Página de información del proyecto
├── css/
│   └── styles.css   # Estilos globales
├── js/
│   └── app.js       # Lógica de la aplicación
└── README.md        # Este archivo
```

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
- Git & GitHub Flow

## Flujo de trabajo (GitHub Flow)

1. Clonar el repositorio
2. Crear una rama por funcionalidad (`feature/nombre`)
3. Desarrollar y hacer commits
4. Subir la rama y abrir Pull Request
5. Revisión y merge a `main`
